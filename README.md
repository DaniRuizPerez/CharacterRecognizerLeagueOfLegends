League of Legends Champion Recognizer
============

This is a computer vision project that my team developed for the Machine Learning course in the junior year of my undergrad in computer science at UDC (Spain). Given an imagen, it tells you the position and name of different League of Legends characters, in different possitions and backgrounds.
